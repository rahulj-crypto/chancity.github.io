# Chancity Sports Club Website

**Ready To Fight** - The official website of Chancity Sports Club, Karnataka's premier Kabaddi team.

## About

Chancity Sports Club is the official Kabaddi team of a G.H.P.S Government School, sanctioned by the Karnataka Sports Federation. This website serves as the digital presence for the club, providing information about the team, tournaments, and registration for the annual Chancity Open Kabaddi Tournament.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Brand Identity**: Consistent use of club name, motto ("Ready To Fight"), and colors
- **Tournament Registration**: Online form for teams to register for tournaments
- **Team Profiles**: Information about coaches and players
- **News Section**: Latest updates, match reports, and announcements
- **Contact Form**: Easy way for visitors to get in touch

## Pages

1. **Home** (`index.html`) - Hero section with motto, about preview, values, and tournament highlights
2. **About Us** (`about.html`) - Mission, values, history, and official affiliations
3. **Teams & Coaches** (`teams.html`) - Coach profiles and player rosters (Senior & Junior)
4. **Tournaments** (`tournaments.html`) - Tournament details, rules, categories, and past winners
5. **Registration** (`registration.html`) - Tournament registration form
6. **News** (`news.html`) - Latest news, updates, and upcoming events
7. **Contact** (`contact.html`) - Contact information and inquiry form

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with CSS variables for brand colors
- **JavaScript** - Interactive features and form validation
- **Font Awesome** - Icons (loaded via CDN)

## Brand Colors

- Primary: Deep Red (`#8B0000`)
- Secondary: Gold (`#DAA520`)
- Dark: Black (`#1a1a1a`)
- Light: White (`#ffffff`)

## Getting Started

1. Clone or download the repository
2. Open `index.html` in a web browser
3. No build process required - static HTML files

## Customization

### Adding Images

Place your images in the `images/` folder. Update the `src` attributes in HTML files. Images have fallback gradients if not found.

### Updating Content

Edit the HTML files directly to update text content. The site is designed for easy content management.

### Changing Brand Colors

Edit the CSS variables in `css/styles.css`:

```css
:root {
    --primary-color: #8B0000;  /* Deep Red */
    --secondary-color: #DAA520; /* Gold */
    /* ... other variables */
}
```
